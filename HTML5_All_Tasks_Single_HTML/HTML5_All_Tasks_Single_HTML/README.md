# HTML5 All Tasks - Single HTML Version

This folder contains HTML5 tasks for the Local Community Event Portal.

## Important Change
Each task now uses a single `index.html` file with CSS inside `<style>` and JavaScript inside `<script>`.
No separate `style.css` or `script.js` files are used.

## Cumulative Method
Each next task includes the previous task code and adds only the new task feature.
For example, Task 5 includes Task 1 to Task 4 code and adds the Event Registration Form.
